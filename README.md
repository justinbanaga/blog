# Description
CRUD blog with comments using Ruby on Rails

# Features
-Create articles <br>
-Add articles <br>
-Edit articles <br>
-Delete articles <br>
-Add comments <br>
-Delete spam comments <br>
-Authentication for Login and Logout functionality <br>
